Decision making statements

In C#, there are conditional/Decision making statements using which we can check for certain condition and based on condition we can do certain task like, we can set a value to variable or write some statement on the console etc. Let's understand this in detail.

The If() statement

The if() statement contains a condition in the parenthesis and if the condition evaluates to true executes a block of code.

if(condition)
{
   //code to execute
}


For example

int i=10;
if(i < 20)
{
   Console.WriteLine("10 is less than 20");
}


Note- The If() statement must contain a boolean condition. Otherwise compiler would throw an error.

The else statement

The else statement executes a block of code if the condition specified in if() statement is evaluates to false.

int i=30;
if(i < 20)
{
   Console.WriteLine("i is less than 20");
}
else
{
   Console.WriteLine("i is not less than 20");
}


output - i is not less than 20

else..if

Suppose you have to check for multiple conditions at once.But using if..else you can't do that.For such scenario, C# has provided else..if statement. Using else..if , you can check for multiple conditions at once.

int i=20;
if(i < 10)
{
   Console.WriteLine("i is less than 10");
}
else if(i < 15)
{
   Console.WriteLine("i is less than 15");
}
else if(i < 30)
{
   Console.WriteLine("i is less than 30");
}
else
{
   Console.WriteLine("Incorrect value");
}


Explanation- Here first if() checks whether i is less than 10, if not , then next if condition checks whether i is less than 15. If true then it will evaluates the block of code otherwise will move to next if condition to check if i is less than 30. It evaluates to true and will print i is less than 30 on the console.


Nested if..else

C# supports if..else statements inside another if..else statement. This is called nested if..else.

if(condition1)
{
   code to execute
   if(condition2)
    {
       code to execute
    }
    else
    {
       code to execute
    }
}
else
{
   code to execute
}


For example

int i=10;
if(i != 10)
{
   if(i < 15)
    {
       Console.WriteLine("i is less than 10");
    }
    else
    {
       Console.WriteLine("i is greater than 10"");
    }
}
else
{
   Console.WriteLine("i is equal to 10"");
}


output - i is less than 10
