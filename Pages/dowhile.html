Do while loop in C#

The do-while loop is similar to while loop except it checks for the condition after the loop executes. It means even if the provided condition evaluates to false at the first time ,loop executes once at least.

Synatx

int i=1;
do
{
   Console.WriteLine(i);
}while(i < 5);


Output
1
2
3
4


In do-while loop even if the condition specified is false , do-while loop will execute once.

int i=1;
do
{
   Console.WriteLine(i);
}while(i < 1);


Output
1

Nested do while

We can also nest a do-while loop i.e do-while inside do-while loop.

int i = 1;
int j = 3;
do
{
   Console.WriteLine(i);
   i++;
    do
    {
     Console.WriteLine(j);
     j--;
    } while (j > 1);
} while (i < 3);


Output
1
3
2
2
1
