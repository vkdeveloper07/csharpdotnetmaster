What is jagged array

Jagged array is an array of array. It means every element of jagged array is a seperate array. The elements of jagged arrays can be of different dimentions.

Syntax

int[][] arr=new int[2][]; jagged array of 2 single dimentional arrays
int[][,] arr=new int[4][,]; jagged array of 4 two dimentional arrays

Initialization

Before we can use the array, it must be initialized
arr[0]=new int[5]{1,2,3,4,5};
arr[1]=new int[3]{5,8,3};

We can also omit the size of array
arr[0]=new int []{3,5,6,1};
arr[1]=new int []{1,9,6};

You can also directly initialize the array while declaring

int[][] jaggedarr=new int[2][]
{
   new int[]{1,2,3,4},
   new int[]{4,1}
};

You can use the short form method to initialize the jagged array

int[][] jaggedarr=
{
   new int[]{1,2,3,4,5},
   new int[]{4,1,4,5}
}

The jagged array is an array of arrays. so the elements of jagged arrays are reference types and initialized to null by default.

Access elements of jagged array

To access the element of the jagged array , specify the row and the index of element of array.

int[][] arr=
{
   new int[]{1,2,3,4},
   new int[]{4,5}
}
Access second([1]) element of first array([0])
Console.WriteLine(arr[0][1]) Output - 2
Access second ([1]) element of second array([1])
Console.WriteLine(arr[1][1]) Output - 5

Jagged array with multidimentional arrays

You can also declare the jagged array with multidimentional arrays. Below is the example

int[][,] arr=new int[2][,]
{
   new int[]{ {2,8},{2,4} },
   new int[]{ {3,6},{4,3} },
}
 Above example declared the two dimentional jagged array with 2 rows and 2 columns.

Now you can access the elements of 2D array as follows
Console.WriteLine(arr[0][1,0]) Output - 2
Console.WriteLine(arr[1][0,1]) Output - 6

Similar to 2D array , you can mix up jagged array with 3d, 4d upto 32 dimentional arrays.
