Switch statement in C#

The switch statement is like if..else where you can specify multiple conditions and based on expression provided, particular block of code is executed.

Syntax

switch(expression/condition)
{
  case value1:
   Code to execute
   break;
  case value2:
   Code to execute
   break;
  default:
   default Code to execute if other cases values does not match the expression.
   break;
}


Explanation- The value or expression provided to switch will be checked against every case. When the value matches with the case value , the respective block of code is executed and the flow of execution is terminated and program jumps out of switch. This is done by break keyword. If you don't specify the break keyword, the execution will continue to next case statement.


Example



Output - The value is 10.


Note: Before c# 6.0, the expression in switch statement must of type char,bool,string,int,long, or enum. But starting witg C# 7, the switch expression could be any non null expression.


The switch values must be unique values. How ever with C# 7 , case can have non unique values as well. In that case , the first matching case will be executed.


Using break to exit the case

After every case statement, you must specify the break keyword to terminate the flow of execution and jump out of switch statement. If you don't specify the break keyword, the flow of execution will continue and the further case statements will be executed.
