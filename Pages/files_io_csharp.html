Working with Files in C#

A file is a collection of data with specific extension stored on a disk. C# provides many classes to work with files. In this module, we will look at some of the important file Classes in provided by C#.

File Classes

FileStream

FileStream class provides the methods and properties to read and write data into the files.
Constructor
FileStream fs=new FileStream("path",FileMode,FileAccess);
Example
FileStream fs=new FileStream("test.txt",FileMode.Open,FileAcess.ReadWrite);

Below table contains the important methods of FileStream class.

Method Description
WriteByte(byte) Writes the byte to the current position in file.
ReadByte() Reades the byte to the from file.
Close() Closes the current stream and release the resources.
Dispose() release the resources used by stream.
Flush() clears the buffer for the stream and write any buffered data to file.
Read(Byte[],int32,int32) Reads the block of byte array and write data to buffer stream.
SetLength(int64) Set the length of stream to given value.
Program

class Program
{
    void Main(string[] args)
   {
      FileStream stream = new FileStream("test.dat",FileMode.OpenOrCreate,FileAccess.ReadWrite);
      for (int i = 1; i <=10; i++)
      {
         stream.WriteByte((byte)i);
      }
   }
}

File

The File is a static class which provides methods and properties to perform operations on physical files.
Below table contains the important methods in file class.

Method Description
Create Creates a file or overrides existing file
Delete Deletes a file
Exists Check if files exists or not
Move Moves the file to specified location with new file name
Open Opens the file for read and write and returns the filestream object
ReadAllText Opens the file and read all the lines and closes the file
ReadAllLines Opens the file and read all the lines and closes the file
ReadAllBytes Opens the file and read the content into byte array and closes the file
WriteAllBytes Create new file and write the byte array into file and closes. If existing file is there , it is overwritten
WriteAllLines Create new file and write the byte array into file and closes. If existing file is there , it is overwritten
WriteAllBytes Create new file and write the strings into file and closes.
WriteAllText Create new file and write the strings into file and closes. If existing file is there , it is overwritten
Program

class Program
{
   static void Main(string[] args)
   {
      if (File.Exists("text.txt"))
      {
         Console.WriteLine("File exists.");
      }
      else
      {
         Console.WriteLine("File does not exists.");
      }
   }
}

StreamReader

StreamReader class contains methods to read characters from byte stream.

Methods Description
Close Closes the streamreader object
Dispose Closes the streamreader and releases the resources
Read Reads the next characters from input stream
ReadLine Reads the line of characters from input stream
ReadToEnd Reads all characters from the current position to the end of stream
Peek Returns the next character from stream
Program

class Program
{
   static void Main(string[] args)
   {
      StreamReader streamReader = new StreamReader("text.txt");
      string line;
      while ((line=streamReader.ReadLine())!=null)
      {
         Console.WriteLine(line);
      }
   }
}

StreamWriter

StreamWriter class contains methods which are used to write characters to stream.

Methods Description
Close Closes the streamwriter object
Dispose writes the buffer data to stream and closes the streamwriter and releases the resources
Flush Clears the buffer for writer and write buffered data to stream
Write write data to stream
WriteLine Write formatted data to stream
Program

class Program
{
   static void Main(string[] args)
   {
      StreamWriter streamWriter = new StreamWriter("test.txt");
      streamWriter.WriteLine("This is test");
      streamWriter.Dispose();
   }
}
