Data Types in C#

C# is strongly typed language. We must declare the variable with appropriate data type. The data type of the variable defines what range of values the variable is going to store.

Example

int i=20;
float amount=34.4f;
string message="Hello";
char character='z';
bool b=false;



Types of Data types

There are mainly two types of data type. Value type and Reference type.

  • Value Type - The value type data type directly stores the value into it's memory.In C#, we have following value type data types.

    Data type Range Description
    Integer type
    byte 8 bit unsigned integer 0-255
    sbyte 8 bit signed integer -128 to 127
    ushort 16 bit unsigned integer 0 to 65535
    short 16 bit signed integer -32768 to 32767
    uint 32 bit unsigned integer 0 to 4,294,967,295
    int 32 bit signed integer -2,147,483,648 to 2,147,483,648
    ulong 64 bit unsigned integer 0 to 18,446,744,073,709,551,615
    long 64 bit signed integer -9,223,372,036,854,775,808 to 9,223,372,036,854,775,807
    Boolean type
    bool 8 bit true or false value true or false
    Floating type
    float 32 bit floating point -3.402823e38 to 3.402823e38
    double 64 bit floating point -1.79769313486232e308 to 1.79769313486232e308
    decimal 128 bit decimal point (+ or -)1.0 x 10e-28 to 7.9 x 10e28
    Character type
    char 16-bit single Unicode character Any character like A,a, or unicode like \u0057
    string type
    string sequence of characters Any valid string enclosed in double quotes like "Hello","Welcome" etc
    DateTime type
    DateTime to represent Date and Time 0:00:00am 1/1/01 to 11:59:59pm 12/31/9999
    Object type
    object Base data type of all types any value can be assigned to object type
    Each Datatype can store specific range of values assigned by compiler as given in above table. If we try to assign value to data type larger then the maximum value, the compiler will throw an error.

Data Type conversion

We can convert values of one data type to another using type conversion. There are two types of type conversion in c#. Implicit and Explicit

  • Implicit - Implicit type conversion is done by compiler automatically. Implicit type conversion happens when
    1. The two data types are compatible.
    2. When small data type value is assigned to large data type.
    Implicit data type conversion table
    From To
    byte short,int,long,float,double
    short int,long,float,double
    int long,float,double
    long float,double
    float double
    char int,long,float,double
  • Explicit type- Some data types are not converted implicitly by compiler. When you try to convert large datatype to small data type , then you have to use explicit type conversion. For ex-

    public static void Main()
    {
        long l= 234;
        int i=(int)l;
    }

    Sometimes casting from long large data types to small datatypes can cause loss of precision.
