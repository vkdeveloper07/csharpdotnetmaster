What is an Array

An array is the data structure in c# which stores multiple elements of the same datatype. These elements are stored contiguously in the memory. For ex- if we want to store single value in variable, we would do so by declarring a variable and assigning value to it. int i=2; But suppose you want to store multiple values in a variable at a time, so we can do so by declaring an array. There are 3 types of array Single dimentional , multi dimentional and jagged array. Let's understand how to declare an array.

Array declaration and Initialization

Declaring an integer array,
int [] array;
Declaring an double array,
double [] array;
Declaring an string array,
string [] array;


Declare the integer array of size 5 i.e 5 elements can be stored in the array,
int[] arr=new int[5];

Declare the string array of size 5
string[] arr=new string[5];

Initializing an integer array
int[] arr=new int[5]{1,2,3,4,5};

Initializing an string array
string[] arr=new string[2]{"Hello","world"};


You can also declare array without specifying size if you are initializing it at the time of declaration. The compiler will determine the size based on the number of elements in the array.

int[] arr={1,2,3,4,5};

Invalid array declaration

Declaring array without specifying size will throw an error.
int[] arr=new int[];

When initializing array , the number of elements must be equal to the size specified in array.
int[] arr=new int[5] {12,,43,76}

Accessing array elements

Array elements can be accessed using
1. Index and
2. loop

Accessing array elements using Index

An index is a number that is associated with each element of array. Array index starts from 0.


For ex

int[] arr=new int[5] {4,1,5,8,9}; Console.WriteLine(arr[1]); output- 1
Console.WriteLine(arr[3]); output- 8

if you try to access the array element specifying index greater than size , compiler will throw an IndexOutOfRange exception.
For ex


int[] arr=new int[5] {4,1,5,8,9}; Console.WriteLine(arr[5]); output- error - IndexOutOfRange

Accessing array elements using loop

int[] arr=new int[5] {4,1,5,8,9};
for(int i=0;i < arr.Length; i++)
{
    Console.WriteLine(arr[i]); output- 4,1,5,8,9
}


Two dimentional Array

Until now we have seen single dimentional array. Here we will see how to declare and initialize two dimentional array.

Declaring and initializing multidimentional array.

int[,] arr =new int[2,3];

Two dimentional array with 2 rows and 3 columns

int[,] arr=new int[2,3]
{
   {2,5,7},
   {4,8,9}
};

Accessing elements of 2 dimentional array

int[,] arr=new int[2,2]
{
   {2,5},
   {4,8}
};
 Console.WriteLine(arr[0,0]); output- 2
Console.WriteLine(arr[0,1]); output- 5
Console.WriteLine(arr[1,0]); output- 4
Console.WriteLine(arr[1,1]); output- 8

Access elements using for loop

int[,] arr=new int[2,2]
{
   {2,5},
   {4,8}
};
for(int i=0;i < 2; i++)
{
  for(int j=0; j < 2 ; j++)
  {
    Console.WriteLine(i,j);
  }
}


Similar to 2 dimentional array , you can declare 3 dimentional , 4 dimentional arrays. C# allows multidimentional arrays upto 32 dimentions.
