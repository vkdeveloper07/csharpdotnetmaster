What is a Class

A class is like a blueprint. Let's take real world example. There are many types of cars like Audi,Mercedese etc. So we can say a car is a class and Audi,Mercedese are object of the car.A car can have many properties like color, speed,shape etc. So the Audi which is object of the car has all of these properties.

Similarly in object oriented programming we can create a class which contains properties like methods,fields, constructors,events ,delegates etc.
