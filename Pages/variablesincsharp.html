Variables in C#

A variable is a name in which we can store the data. A variable is declared prefixing data type of variable.

Syntax

< datatype > < variable name > =value;

Example

int age=20;
float amount=23.2;



Naming conventions

  • Variable names must be unique throught the program.
  • Variable name should contain letters,digits and underscores only.
  • Variable names are case sensetive.
  • Reserved keywords cannot be used as variable names.
  • Variable name should start with letter.

Declaration and initialization

We can declare and initialize variable on the same line.
int i=20;
OR
we can declare variable first and initialize later. int i;
i=20;

We can declare multiple variables of same data type on single line.
int i , j, k;
AND
We can initalize multiple variables of same data type on single line.
int i=10 , j=20, k=100;

We can assign value of one variable to another.
int i = 30;
int j=i; //Now j contains value of i i.e 30

We can assign another value to variables after it has been initialized.
int i=20;
i=30;
Console.WriteLine(i); //value if i is 30.

A variable must be assigned value before using it. Otherwise compiler will throw an error.
int i;
Console.WriteLine(i); //error- Use of unassigned local variable i;
