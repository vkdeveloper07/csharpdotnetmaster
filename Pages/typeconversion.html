What is Type conversion

Type conversion means converting value of one datatype to another datatype.If the two datatypes are compatible, c# automatically converts the value of data types. If two datatypes are not comparible, you have to externally specify the data type manually. This is called type casting.

Types of Type conversion

There are basically two ways in which we can convert values between data types.

  • Implicit Type conversion - happens when two data types are compatible. So compiler automatic does the type conversion.
  • Explicit Type conversion - when do data types are not compatible , so you need to manually convert the types.

Implicit type conversion

C# compiler can handle implicit type conversion between numberic data types. For ex- we can assign byte value to short, int, long, float and double as the size of these data types are larger than byte, so compiler can automatically convert the value without truncating the value.
The following list shown the what data types conversions can be done implicitly.

Byte -> Short -> int -> long -> float ->double
char -> int -> long -> float ->double
Short -> int -> long -> float ->double
int -> long -> float ->double
long -> float ->double
float ->double

Explicit type conversion

When two data types are not compatible, the compiler wants you to explicitly specify the type in which to convert the value. This is known as cast. For ex- if you try to convert double value to int, without explicitly specifying, the compiler will throw an error.
To convert explicitly, specify the data type.
For ex-
double d=123.4
int i=(int)d;
output - 123
