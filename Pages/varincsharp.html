var in C#

var is introduced in C# 3.0. We can declare method level variables using var. The var is very useful for developers. It allows developers to declare the variables without specifying their type. The data type of variable will be decided based on value assigned to it.

Declaration and Initialization

var age=20;
var msg="Hello";
var amount=20.3;

In case of var , the compiler will decide the type of variable based on the value assigned to it. See the following program.



variables with var datatype must be initialized at the time of declaration. Otherwise compiler will throw an error.

var i;//error- implicit typed variables must be initialized
i=20;



Multiple declaration of variables with var datatype are not allowed in single statement.

var i=20,j=10; //will throw error


var cannot be used as function parameter

public void Show(var i) error

where can we use var

  • var can be used to declare implicitly typed local variables.
  • var can be used to declare anonymous types.
  • var can be used with LINQ queries.
  • var can be used in for and foreach loops.
