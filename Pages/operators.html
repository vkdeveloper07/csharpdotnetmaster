Operators in c#

Operators in C# are special symbols that perform special type of operation on operands.For example, int i=2; So = is an assignment operator which assigns value 2 to variable i. Let's discuss the operators in detail.

Types of operators

There are basically two types of operators in C#. Unary and binary operators.

  • Unary operators- Unary operators operate upon single operand. For ex- int i=8;Console.WriteLine(+i) Here '+' is a unary operator applied to 'i' returning its value.
  • Binary operators- Binary operators operate open two operands. For ex- int a=2,b=4; Console.WriteLine(a+b); Here binary '+' performs the addition of value of variables 'a' and 'b' and prints on console.

Relational Operators
< less than operator Checks if left operand is less than right operand a < b
> greater than operator Checks if left operand is greater than right operand a > b

Bitwise Operators

<< Bitwise left shift operator Shifts the bits to the left a << b
>> Bitwise right shift operator Shifts the bits to the right a >> b

%= Modulus Assignment operator Calculate reminder of division of the left and right operand and stored value in left operand a %= b (a=a%b)
