Structure in C#

In C#, structure is a value type data type. Like classes , structure can contain parameterized and static constructors, methods, fields ,properties events , operators etc. Structure is used to hold small data type values that do not require modification after structure is created.

Structure is created with struct keyword. Let's understand how to declare and use structure.

Structure declaration

struct struct_name
{
   //constructors
   //properties
   //Methods
}


Example

struct Student
{
   public string name;
   publlic int age;
}

Creating object of structure

An object of structure can be created with new keyword.
Student std=new Student();
When the structure object is created with new keyword, the default parameterless constructor of the structure is called and all the members are initialized to default value.

You can also create object of structure without new keyword ,
Student std;
but then you must initialize the members of the structure manually. If you try to access value of member of strucure without initializing it , it will throw an error.

Access members of structure

Like classes you can access the members of structure using the dot(.) operator.
Console.WriteLine(std.age);

Constructors in structure

a structure can contain parameterized constructor, but not parameterless constructor.

Example

struct Student
{
    public int age;
    public Student(int a)
    {
     age=a;
    }
}
Student std=new Student(12);
Console.WriteLine(std.age);


Methods in structure

Structure can include methods like classes.


struct Student
{
    public int age;
    public Student(int a)
    {
     age=a;
    }
    public void Show()     {
       Console.WriteLine(this.age);
    }
}
Student std=new Student(12);
std.Show();

Difference between class and structure

Class Structure
class is defined using class keyword. structure is defined using struct keyword.
class is reference type data type. structure is value type data type.
class supports inheritance. structure does not support inheritance.
class can have parameterless constructors. structure cannot have parameterless constructors.
An object of class can be created with new keyword. An object of structure can be created with or without new keyword.
class can inherit other classes. structure cannot inherit other classes or structure.
class can be used as a base class for other classes. structure cannot be used as base structure for other strucures.
We can declare classes as abstruct, sealed, virtual and protected. We cannot declare structures as abstruct, sealed, virtual and protected.
