Introduction To C#

The C# is multi paradigm, object oriented,general purpose programming language. It was developed by a team which was led by Anders Heijlsberg at Microsoft in 2000.C# is very flexible and powerful. It provides full support to object orientation programming approach.C# is also type safe programming language. C# is used to create variety of applications.

Features of C#

  • C# is very simple to learn and understand.
  • C# is designed to be a modern programming language.
  • C# is very strongly typed.
  • C# syntax is similar to most of the object oriented programming languages.
  • C# is also used for game development.
  • C# is used to create robost and secure applications.

Applications of C#

  • Web Services.
  • Windows form applications.
  • Web application .
  • Windows Services.
  • Class Libraries.
  • Game development.
