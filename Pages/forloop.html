For loop

The for loop is also like while and do-while loop. It executes a block of code multiple times until the condition specified becomes false. The for loop is different in syntax compared to while and do-while loop.

Syntax

for(initialization;condition;increment)
{
   Code to execute
}


  • Initialization- initialize the variable to the value. For ex- int i = 1;
  • Condition - specify the condition for the loop to execute.
  • Increment - increment the value of variable.

Example


for(int i=1;i < 5;i++)
{
    Console.WriteLine(i);
}


Output -
1
2
3
4

You can also specify multiple expressions in for loop

for(int i=1, j=5 ;i < 5;i++,j--)
{
    Console.WriteLine(i + " " + j);
}


Output
1   5
2   4
3   3
4   2


Nested for loop

C# allows nesting of for loops. i.e for loop inside for loop

for(int i=1;i < 3;i++)
{
    for(int j=i;j < 5;j++)
    {
      Console.WriteLine(i + " " +j);
    }
}


Output
1   5
2   4
3   3
4   2

Omitting initialization, condition and increment

You can omit initialization, condition and increment from the loop. For ex , you can initialize variable before the loop. Specify condition and increment inside the loop.

int i = 1;
for (; ; )
{
   if (i < 5)
   {
     Console.WriteLine(i);
     i++;
   }
   else
     break;
   }


Output
1
2
3
4
