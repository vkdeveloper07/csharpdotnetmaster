What is an exception

An exception is a problem that prevents a program from executing successfully. When an exception occures, the program crashes and execution is halted.

Exceptions handling

The exceptions must be handled properly to avoid program crashing. C# provides built in mechanism to handle the exception.
We can use the built in keywords try, catch and finally to handle the exception occured in program.

Syntax

try
{
   //code in which exception can occur
}
catch(Exception e)
{
   //code to handle the exception
}
finally
{
   //final code to execute
}

Example

try
{
   int i=10/0;
}
catch(Exception e)
{
   Console.WriteLine(e.Message);
}
finally
{
   Console.WriteLine("Error catched");
}

Explanation

Try

In try block , we write the code in which exception can occur.When the error occures , the program control is thrown to first matching catch block.

catch

In catch block , we handle the error that was thrown from the catch block.

finally

The finally block is executed despite exception is raised on not. We can write the code to clear up resources and memory space in finally block. finally block is optional.

Multiple catch block

A try catch can have multiple catch blocks.The first catch block which matches the exception thrown by try block will be executed.

try
{
   int i=10/0;
}
catch(DivideByZeroException e)
{
   Console.WriteLine(e.Message);
}
catch(Exception e)
{
   Console.WriteLine(e.Message);
}
finally
{
   Console.WriteLine("Error catched");
}

In above program, the first catch block is executed with DivideByZeroException and prints the exception on console.
Please note , we cannot declare multiple catch block with the same exception type specified. The catch block with exception type Exception must be last catch block.

Nested try catch

C# allows nested of try..catch block. i.e try..catch inside another try..catch

try
{
   try
   {
       int i=10/0;
   }
   catch(DivideByZeroException e)
   {
      Console.WriteLine(e.Message);
   }
}
catch(Exception e)
{
   Console.WriteLine(e.Message);
}
finally
{
   Console.WriteLine("Error catched");
}

In above program, the try..catch block is in outer try block. It throws and catches the exception and prints the message on the console.
