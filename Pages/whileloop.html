What are loops in C#.

A meaning of loop is repeating something again and again. Similarly in C# , loops are used to execute a statement continuously until certain condition is met. Also loops are used to iterate over Array or collection. Instead of accessing each value of array or collection manually index by index we can loop over it and access each element. we will learn about Arrays and collections in next modules.

While Loop

A while loop accepts a condition and executes a block of code until that condition becomes false.


Syntax

while(condition)
{
   Code to execute
}

Like if..else the while loop must contain a boolean expression that either evaluates to true or false. It has three steps. Initialization, condition and increment. Let understand it with below example.


int k=1; //Initialize variable k.
while(k <= 5) //check against condition
{
   Console.WriteLine(k);
   k++;//increment the value
}


Output -
1
2
3
4
5.


Make sure the condition specified in while loop becomes false at some point , otherwise loop will execute infinitly.

Nested while loop

We can write while loop inside another while loop. This is called as nested while loops.

For example

int k=1;
int j=5;
while(k <= 5)
{
   Console.WriteLine(k);
   while(j>=1)
   {
      Console.WriteLine(j);
      j--;
   }
   k++;
}


Output -
1
5
4
3
2
1
2
3
4
