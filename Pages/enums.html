What are Enums

In c#, enum is used to assign the constant name to the integer values. Enum is a group of named integer constants. Enums are used to give values a constant name and make it more readable.

Enum is a value type data type.It can be directly defined inside a namespace , class or structure.

Syntax

enum enum_name
{
    constant_names
}

Example

enum Fruits
{
   Apple,
   Kiwi,
   Banana,
   Pineapple
}


Each constant in enum stands for an integer value starting with 0. The values increases by 1 subsequently for the next constants. For ex , In enum Fruits, the Apple will have value 0, the Kiwi will have value 1 and so on.

If you want to assign some other values to enum constants, you can assign that externally. For ex , you can say Apple =1 , Kiwi=2 etc. If you assign different value to first element of enum, the value will increase by 1 for other elements. For ex - If you assign Apple = 2, the Kiwi will have value of 3, the Banana will be 4 and Pineapple will have value of 5 and so on

Access enum value

To access the enum, use enum_name.constant_name.

Console.WriteLine(Fruits.Apple) output -Apple
Console.WriteLine(Fruits.Kiwi) output -Kiwi


To access the integer value of enum , use enum_name.constant_name and simply type cast it to integer.

Console.WriteLine((int)Fruits.Apple) output -0
Console.WriteLine((int)Fruits.Banana)output -2


You can declare enum of any numeric type. i,e byte , int, short etc. You cannot declare enum of string type.
