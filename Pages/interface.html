What is an interface in c#

An interface is like a defined contract between the two or more classes. Interface contains methods, properties and events. All the methods declared in the interface are abstract methods i.e they cannot have implementation but only declaration. Interface are used to achieve multiple inheritance.
An interface is defined using interface keyword.
The classes or structures must impliment the interface and define all the methods of the interface.


Syntax

interface interface_name
{
   //Methods declaration
   //properties declaration
   //events declaration
}


Example

interface IStudent
{
   void ShowName();
   void ShowAge();
}


All the members of the interface are public by default. you cannot use any other access modifier on members of the interface. If you specify any other access modifier, the compiler will throw an error.

Implementing interface

The classes and structures can implement the interface :

Example

interface IStudent
{
   void ShowName();
   void ShowAge();
}

class StudentDemo:IStudent
{
   public void ShowName()
   {
      Console.WriteLine("Student name is ABC");
   }
   public void ShowAge()
   {
      Console.WriteLine("Student age is 12");
   }
}

The methods of the interfaces must be implemented with public access modifier.


In above program , we have implemented interface implicitly. We can also implement interface explicitly.

class StudentDemo:IStudent
{
   void IStudent.ShowName()
   {
      Console.WriteLine("Student name is ABC");
   }
   void IStudent.ShowAge()
   {
      Console.WriteLine("Student age is 12");
   }
}

In explicit interface implementation ,call the member name with interface name and dot(.). An interface should be implemented explicitly when class is implementing multiple interfaces.
